# RECALL

Portable AI memory on Supermemory Local — import your ChatGPT, Claude, Grok, and coding-agent history into one structured brain, then inject it everywhere.

## Quick start

```bash
# 1. Start local memory engine
npx supermemory local

# 2. Run the app
cd app
npm install
npm run dev
```

Open `http://localhost:3020` (or your deployed base path).

## Structure

- `landing/` — static marketing site
- `app/` — Next.js dashboard (mock API; backend wires in behind `lib/mock-api.ts` contract)
